# Part I: IP Philosophy Under MDA

## 1.1 IP as System Property

Under MDA, intellectual property protection is not an afterthought — it is a **fundamental business asset** that must be:
- **Documented from inception** (not retrofitted)
- **Validated continuously** (not once at launch)
- **Aligned with CPT** (IP strategy serves product vision)
- **Defensible legally** (not just technically clever)

**IP Rule**: Every system evolution pass must include IP impact assessment.

---

## 1.3 IP-Centric Thinking

IP validation starts with:
1. **Invention Identification** — What's novel here?
2. **Prior Art Search** — Has this been done before?
3. **Risk Assessment** — Are we infringing on others' IP?
4. **Protection Strategy** — Patent, trade secret, or open source?
5. **Documentation** — Can we prove our innovation timeline?

Conduct full IVA when:
- **New feature development** (novel techniques introduced)

# Part IV: IP Protection Strategies

## 4.1 Patent Strategy Assessment

### When to Patent vs. Trade Secret

```markdown
## Patent vs. Trade Secret Decision Matrix

| Factor | Patent | Trade Secret |
|--------|--------|--------------|
| **Discoverability** | Easily reverse-engineered | Hard to discover from product |
| **Duration** | 20 years max | Indefinite (if maintained) |
| **Cost** | High ($15K-$50K+) | Low (security measures) |
| **Public Disclosure** | Required | Not disclosed |
| **Enforcement** | Sue infringers | Misappropriation claims |
| **Examples** | Algorithms, methods | Source code, customer data |

### Patent Strategy Recommendations

**High-Value Patents (File):**
1. [Innovation 1]: Core competitive advantage
   - Patentability: High
   - Business Impact: Critical
   - Defensive Value: High
   - Filing Priority: Immediate

**Trade Secret Protection (Don't File):**
1. [Innovation 2]: Implementation details
   - Patentability: Medium
   - Business Impact: Medium
   - Secrecy Maintainable: Yes
   - Protection Method: NDAs, access control

**Defensive Publication (Disclose):**
1. [Innovation 3]: Good idea, not worth patenting
   - Purpose: Block competitors from patenting
   - Method: Publish in IP.com, GitHub, blog
   - Benefit: Freedom to operate

**Open Source Contribution (Release):**
1. [Innovation 4]: Industry standard enhancement
   - Purpose: Drive adoption, build ecosystem
   - License: Apache 2.0 recommended
   - Patent Grant: Include explicit grant
```

# Part VII: IP Anti-Patterns

## Anti-Pattern 1: Invention Amnesia
Failing to document innovations as they're created.

**Resolution**: File invention disclosures immediately upon reduction to practice.

---

## Anti-Pattern 2: License Blindness
Adding dependencies without checking licenses.

**Resolution**: Automated license scanning in CI/CD pipeline.

---

## Anti-Pattern 3: Patent Trolling Yourself
Over-patenting obvious techniques or standard practices.

**Resolution**: Focus on high-value, defensible innovations only.

---

## Anti-Pattern 4: Trade Secret Leakage
Failing to protect trade secrets with proper security.

**Resolution**: Access controls, NDAs, clean room procedures.

# Part VIII: IP Compliance Frameworks Reference

## Software Patent Eligibility (Alice Test)

In the US, software patents must pass the Alice/Mayo test:

**Step 1**: Is the claim directed to an abstract idea?
- If no → Patentable
- If yes → Go to Step 2

**Step 2**: Does the claim contain an "inventive concept"?
- If yes → Patentable
- If no → Not patentable

**Tips for Patent Eligibility**:
- Tie innovation to specific technical problem
- Show concrete improvement to computer functionality
- Avoid pure business methods or mental processes
- Include implementation details, not just the idea

## Inventor Rights by Jurisdiction

### United States
- First to file (since 2013)
- Grace period: 1 year from public disclosure
- Patent term: 20 years from filing

### Europe (EPO)
- First to file
- No grace period (strict novelty requirement)
- Software patents allowed if "technical effect"

### China (CNIPA)
- First to file
- Grace period: 6 months (limited circumstances)
- Increasing importance for global protection
